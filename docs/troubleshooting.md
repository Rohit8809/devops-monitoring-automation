
ubleshooting Guide

## GitHub

### Invalid username or token

Cause:
PAT token issue.

Solution:
Configure SSH authentication.

---

## Permission denied (publickey)

Cause:
SSH key not configured.

Solution:
Add SSH public key to GitHub.

---

## SMTP Errors

### Host doesn't allow connection

Cause:
Incorrect Gmail settings.

Solution:
Use App Password and SMTP port 465.

---

## Alertmanager

### No alerts field found

Cause:
Grafana payload format mismatch.

Solution:
Modify Flask middleware parsing.

---

## osTicket

### Maximum open tickets reached

Cause:
Ticket limit reached.

Solution:
Close low-priority tickets or increase limits.

---

## HTML Email Formatting

Cause:
Plain text email template.

Solution:
Enable HTML email templates.
